# VibeCodingTest_DreamoonTech_Shaun

## 1. Background

- 1.1 Users want this AI platform to help them create new resumes and cover letters.
- 1.2 Users want this AI platform to help them imporve their uploaded resumes and cover letters.
- 1.3 Users expect this AI platform to collect crucial information from their resumes, and provide some job-search suggestions.
- 1.4 This is not a free AI platform, and it requires monetization methods.

## 2. Tech Stacks

- 2.1 Full stack framework: Next.js 14 (App Router)
- 2.2 Validation: Clerk
- 2.3 User Interface: Tailwind
- 2.4 Database: Supabase
- 2.5 Payment: Stripe subscriptions (test mode)
- 2.6 Deployment: Vercel

## 3. Main function Implementation

- 3.1 Using Claude Code API to do the resume analysis.
