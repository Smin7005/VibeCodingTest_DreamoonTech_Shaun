# VibeCodingTest_DreamoonTech_Shaun

## 1. Background

- 1.1 Users want this AI platform to help them create new resumes and cover letters.
- 1.2 Users want this AI platform to help them imporve their uploaded resumes and cover letters.
- 1.3 Users expect this AI platform to collect crucial information from their resumes, and have some job-search suggestions.
- 1.4 This is not a free AI platform, and it requires monetization methods.

## 2. Target

- 2.1 Access restriction of several advanced functions will be applied to free users in this AI platform.
- 2.2 This AI platform will mainly focus on resume improvement.
- 2.3 This AI platform should try to increase the purchasing rate of users.
- 2.4 Operation and interface of this AI platform must be easy and convenient to users.

## 3. Non-target

- 3.1 This AI platform should not have the function about recruitment and related implementation.
- 3.2 This AI platform should not have functions about community services.
